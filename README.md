# electron-google-custom-app
Mecanismo de busca personalizada do Google com Electron

## Install
```
 git clone https://github.com/ClubeDosGeeksCoding/electron-google-custom-app.git
```
Install packages
```
npm install
```
## Run
```
npm start
```
